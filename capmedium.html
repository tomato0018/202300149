Cap Material (마개 재질)

Main Column (항목) 내용 Description
Term Name 마개 재질 Cap Material
Definition (정의) 향수 용기의 입구를 덮는 마개를 제작하는 데 사용되는 물질의 종류를 의미한다. Refers to the type of material used to create the cap that covers the opening of the perfume bottle.
Comment (주기) 향수 용기의 입구를 덮는 마개를 제작하는 데 사용되는 물질의 종류를 의미한다. There are glass, plastic, ceramic, porcelain, metal, stainless steel, and recycled materials.
Requirment (필수) 선택 Option
Repeatable (반복) 반복 불가 Nonrepeatable
Search (검색) 검색 불가 No
