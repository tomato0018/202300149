Container Material (용기 재질)

Main Column (항목) 내용 Description
Term Name 성분 Container Material
Definition (정의) 향수를 담는 용기를 제작하는 데 사용되는 재료의 종류를 의미한다. Refers to the type of material used to create the container that holds the perfume..
Comment (주기) 유리, 플라스틱, 세라믹, 도자기, 금속, 스테인리스 스틸, 재활용 재료 등으로 기록할 수 있다. It can be recorded as glass, plastic, ceramic, ceramic, metal, stainless steel, recycled materials, etc.
Requirment (필수) 선택 Option
Repeatable (반복) 반복 불가 Nonrepeatable
Search (검색) 검색 불가 No
Vocabulary (어휘) Perfume Terms
