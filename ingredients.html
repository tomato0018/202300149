Ingredients (성분)

Main Column (항목) 내용 Description
Term Name 성분 Ingredients
Definition (정의) 향수를 제조할때 사용되어 구성되어있는 물질을 의미한다. Refers to the substances used to create and compose the perfume.
Comment (주기) 에센셜 오일(Essential Oil), 향료(Fragrance Compounds), 알코올(Alcohol), 정제수(Distilled Water), 고정제(Fixatives)등이 있다. There are essential oils, fragrance compounds, alcohol, distilled water, and fixatives.
Requirment (필수) 필수 Mandatory
Repeatable (반복) 반복 불가 Nonrepeatable
Search (검색) 검색 가능 Yes
