Term Name: 용량 (Volume) More Details
URI https://tomato0018.github.io/202300149.github.io/volume.html
Definition 향수 용기 내부에 포함된 향수의 총 부피를 의미하며, 경우에 따라 무게로도 표시될 수 있다.
Comment 용량은 밀리미터(ml), 온스(fl.oz), 그램(g) 등 적절한 단위를 사용하여 기술한다.

External Terms 외부 요소

Term Name: 제품명 (Name)
URI http://purl.org/goodrelations/v1#name
Definition 각 리소스를 설명하는 짧은 텍스트다.
Comment 이 속성은 의미론적으로 dcterms:title 및 rdfs:label과 동등하며 데이터를 표시하는 편리한 바로 가기를 의미한다.
Term Name: 제조사 (Brand)
URI http://schema.org/brand
Definition 제품 또는 서비스와 관련된 브랜드 또는 조직 또는 사업가가 유지 관리하는 브랜드다.
Term Name: 제조국 (Location)
URI http://purl.org/dc/terms/Location
Definition 공간 영역 또는 명명된 장소다.
Term Name: 설명 (Description)
URI http://www.heppnetz.de/ontologies/goodrelations/v1#description
Definition 자원에 대한 짧은 텍스트 설명이다.
Comment 이 속성은 의미론적으로 rdfs:comment와 동일하며 데이터를 표시하는 편리한 바로 가기를 의미한다.
Term Name: 대상 (has Target)
URI http://ns.inria.fr/provoc#hasTarget
Definition 제품 또는 서비스의 주된 타깃이다.
Comment 남성, 여성, 공용으로 구분하여 기술한다.
Term Name: 사용 기한 (Period of Time)
URI http://purl.org/dc/terms/PeriodOfTime
Definition 향수가 변질되지 않은 상태로 사용할 수 있는 기간이다.
Comment 개봉 후(after open), 제조일자로부터(from date of manufact) 등의 문구를 활용해 기술한다..
Term Name: 추가 속성-판매 유형 (additionalProperty)
URI https://schema.org/additionalProperty
Definition 제품의 판매 유형이다.
Comment 판매품(For sale) 또는 비매품(Not for sale) 중 한 가지로 기술한다.
Term Name: 가격 (Price Specification)
URI http://purl.org/goodrelations/v1#PriceSpecification
Definition 자원에 책정된 가격을 기술한다.
Comment 향수의 가격으로 화폐 단위와 함께 아라비아 숫자로 정가를 기술한다. 화폐단위는 원(￦)을 기본으로 하되, 용기에 표기된 단위나 자국의 화폐단위로 기록할수 있다.
Term Name: 출처 (Source)
URI http://purl.org/dc/terms/source
Definition 기술된 자원이 파생된 관련 자원을 기술한다.
Comment 향수의 공식 구매처를 기본으로, 온-오프라인의 다양한 구매처를 기록할 수 있다.
