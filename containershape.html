Container shape (용기 형태)

Main Column (항목) 내용 Description
Term Name 용기 형태 Container shape
Definition (정의) 향수를 담는 용기의 물리적 구조와 디자인을 의미한다. Refers to the physical structure and design of the container that holds the perfume.
Comment (주기) 유리(Glass), 플라스틱(Plastic), 세라믹(Ceramic), 도자기(porcelain) , 금속(Metal), 스테인리스 스틸(Stainless Steel), 재활용재료(Recycled Materials)등으로 기록할 수 있다. Refers to the physical structure and design of the container that holds the perfume.
Requirment (필수) 선택 Option
Repeatable (반복) 반복 불가 Nonrepeatable
Search (검색) 검색 불가 No
